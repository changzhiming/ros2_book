旋转平移矩阵

二维

顺时针旋转

[ x ′ y ′ ] = [ cos ⁡ θ sin ⁡ θ − sin ⁡ θ cos ⁡ θ ] [ x y ]

旋转平移

T = [ cos ⁡ θ − sin ⁡ θ t x sin ⁡ θ cos ⁡ θ t y 0 0 1 ]

计算

x ′ = x cos ⁡ θ − y sin ⁡ θ + t x

y ′ = x sin ⁡ θ + y cos ⁡ θ + t y

pid.md

PID 控制器

u ( t ) = K p , e ( t ) + K i ∫ 0 t e ( τ ) , d τ + K d d e ( t ) d t

各部分作用

部分 符号 功能 直观理解
比例 (P) Kp * e(t) 根据当前误差产生控制输出 误差越大，输出越大，用于快速纠正偏差
积分 (I) Ki * ∫ e(t) dt 累积过去的误差，消除静态偏差 解决长期稳态误差（steady-state error）
微分 (D) Kd * de(t)/dt 根据误差变化速度预测未来趋势 抑制过冲和振荡，使系统更平滑 
#include "rclcpp/rclcpp.hpp"
#include <vector>

class PIDController
{
public:
    PIDController(double kp, double ki, double kd)
        : kp_(kp), ki_(ki), kd_(kd), integral_(0.0), prev_error_(0.0) {}

    double update(double target, double current, double dt)
    {
        double error = target - current;

        // 比例
        double P = kp_ * error;

        // 积分
        integral_ += error * dt;
        double I = ki_ * integral_;

        // 微分
        double derivative = (error - prev_error_) / dt;
        double D = kd_ * derivative;

        prev_error_ = error;

        return P + I + D;
    }

private:
    double kp_, ki_, kd_;
    double integral_;
    double prev_error_;
};

调试技巧

参数 调整原则
Kp 先调比例，让系统快速响应
Ki 调积分，消除稳态误差，但注意可能产生振荡
Kd 调微分，减少过冲，提高系统稳定性
注意 PID 参数调节需结合采样周期 dt，否则积分和微分计算可能不准确
